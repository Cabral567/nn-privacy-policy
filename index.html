NN Drive Upload
Extensão para Google Chrome

Política de Privacidade

Última atualização: abril de 2026

Resumo: Esta extensão não coleta, armazena nem compartilha dados pessoais em servidores próprios. Toda a operação ocorre diretamente entre o navegador do usuário e os servidores do Google. Nenhum dado é enviado a terceiros.

1. Informações gerais

A extensão NN Drive Upload foi desenvolvida pela NN Contabilidade & Consultoria para uso interno da equipe. Esta política descreve como a extensão coleta, usa, processa, armazena e compartilha dados do usuário, em conformidade com a Google API Services User Data Policy e as Chrome Web Store Policies.

2. Dados coletados

A extensão acessa os seguintes dados via API do Google, com autorização explícita do usuário:

DadoFinalidadeArmazenamento
Nome completoExibição na interface para confirmar a conta conectadaNão armazenado
Endereço de e-mailExibição na interface da extensãoNão armazenado
Foto de perfilExibição como avatar na interfaceNão armazenado
Arquivos enviados pelo usuárioUpload para o Google Drive do próprio usuárioApenas no Drive do usuário

Nenhum dado é enviado para servidores da NN Contabilidade & Consultoria ou de terceiros.

3. Como os dados são usados

Os dados acessados são utilizados exclusivamente para:

  • Autenticar o usuário com sua conta Google via OAuth2
  • Exibir o nome, e-mail e foto do usuário conectado na interface
  • Criar pastas no Google Drive do usuário
  • Fazer upload dos arquivos selecionados para o Drive do usuário
  • Configurar permissões de acesso (público ou restrito ao proprietário)
  • Gerar links de visualização dos arquivos enviados

4. Processamento e armazenamento

Sem servidor próprio. A extensão opera inteiramente no navegador do usuário. Os arquivos trafegam diretamente do navegador para a API do Google Drive — nenhum dado passa por infraestrutura da NN Contabilidade & Consultoria.

Token de autenticação: gerenciado pelo próprio Chrome via chrome.identity. Não é persistido em banco de dados ou servidor externo.

Dados de perfil: nome, e-mail e foto são carregados a cada sessão e mantidos apenas na memória do popup. Não são gravados em nenhum local.

5. Compartilhamento de dados

A extensão não compartilha nenhum dado com terceiros. As únicas comunicações externas são com serviços oficiais do Google:

  • Google OAuth2 API — autenticação do usuário
  • Google Drive API v3 — criação de pastas, upload e permissões

Essas comunicações estão sujeitas à Política de Privacidade do Google.

6. Escopos OAuth utilizados

EscopoMotivo de uso
drive.fileCriar pastas e fazer upload de arquivos no Drive do usuário. Acesso restrito apenas aos arquivos criados pela própria extensão — não concede acesso a arquivos preexistentes.

7. Documentos confidenciais

Quando o usuário ativa a opção "Documento confidencial", o arquivo é salvo em subpasta restrita e a extensão remove ativamente qualquer permissão de acesso público, garantindo que apenas o proprietário da conta Google autenticada tenha acesso.

8. Retenção de dados

A extensão não retém dados. Os arquivos enviados ficam no Google Drive do próprio usuário, sob controle total dele. O usuário pode excluí-los a qualquer momento diretamente no Drive.

9. Direitos do usuário

  • Revogar acesso: em myaccount.google.com/permissions a qualquer momento
  • Excluir dados: arquivos no Drive podem ser excluídos diretamente pelo usuário
  • Desinstalar: encerra todo o acesso imediatamente

10. Alterações nesta política

Alterações serão publicadas nesta página com data de atualização revisada. O uso contínuo da extensão constitui aceite das novas condições.
