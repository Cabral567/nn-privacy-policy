NN Drive Image Drop
Extensão para Google Chrome

Política de Privacidade

Última atualização: abril de 2026

Resumo: Esta extensão não coleta, armazena nem compartilha nenhum dado pessoal. O único acesso é ao Google Drive do próprio usuário, com permissão explícita e restrita.

1. O que esta extensão faz

A extensão NN Drive Image Drop permite que o usuário envie imagens diretamente para uma pasta no seu próprio Google Drive e obtenha um link de visualização. O fluxo completo ocorre entre o navegador do usuário e os servidores do Google — nenhum dado passa por servidores da NN Contabilidade & Consultoria.

2. Dados acessados

Para funcionar, a extensão solicita autenticação com a conta Google do usuário. Através dessa autenticação, acessa:

  • Nome e e-mail — exibidos na interface da extensão para confirmar qual conta está conectada.
  • Foto de perfil — exibida como avatar na interface.
  • Google Drive — somente para criar uma pasta chamada "Drive Image Drop" e salvar os arquivos enviados pelo próprio usuário.

O escopo OAuth utilizado é drive.file, o mais restrito disponível — que permite acesso apenas aos arquivos criados pela própria extensão, sem acesso a nenhum outro arquivo do Drive do usuário.

3. O que não fazemos

  • Não coletamos nenhum dado pessoal
  • Não armazenamos arquivos, tokens ou informações em servidores próprios
  • Não compartilhamos dados com terceiros
  • Não monitoramos o uso da extensão
  • Não acessamos histórico de navegação, abas ou outros dados do navegador

4. Armazenamento do token de autenticação

O token de autenticação do Google é gerenciado exclusivamente pelo próprio navegador Chrome, através da API chrome.identity. A extensão não persiste o token em nenhum local externo.

5. Permissões declaradas

  • identity — necessária para autenticar o usuário com a conta Google via OAuth2.

6. Serviços de terceiros

A extensão se comunica exclusivamente com as APIs oficiais do Google:

  • Google Drive API (upload de arquivos e criação de pasta)
  • Google Drive About API (nome, e-mail e foto do usuário logado)

Estas comunicações estão sujeitas à Política de Privacidade do Google.
